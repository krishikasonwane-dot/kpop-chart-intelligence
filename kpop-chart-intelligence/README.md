# 🎵 K-Pop Chart Intelligence Dashboard

A machine learning and analytics platform designed to analyze:
- Chart re-entry behavior
- Comeback momentum
- Fandom-driven engagement
- K-Pop streaming dynamics

## 🚀 Features

- Re-entry detection engine
- Momentum spike analysis
- Fandom intensity scoring
- Machine learning prediction model
- Interactive Streamlit dashboard

## 🛠️ Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
- Scikit-Learn

## 📊 KPIs

- Re-Entry Frequency
- Momentum Spike Score
- Fandom Intensity Proxy
- Rank Recovery Speed

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run dashboard/app.py
```

## 🤖 Machine Learning

Model:
- Random Forest Classifier

Goal:
- Predict chart re-entry probability
